## AI-Powered Career Navigation Platform

A comprehensive web application designed to help students discover their true potential through AI-driven career guidance, gamified learning paths, and realistic career simulations.

### Features

- **Self-Discovery Tests**: 5-stage comprehensive assessment covering personality, analytical skills, communication, emotional intelligence, and game simulation
- **AI-Powered Matching**: Personalized career recommendations based on individual strengths and interests
- **Gamified Learning**: Interactive quests, badges, and XP system to make learning engaging
- **Career Simulation**: Real-world scenarios to test decision-making and role-specific skills
- **Parent Support**: Dedicated section to help parents understand and support their child's journey
- **Free Platform**: Complete access to all features at no cost

### Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run the Application**:
   ```bash
   npm start
   ```

3. **Open in Browser**:
   Navigate to `http://localhost:3000`
